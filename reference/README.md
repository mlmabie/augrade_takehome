# reference/

Supporting documents that live alongside the runnable pipeline.

- [`experiments/`](experiments/) — artifacts produced during this task: memos, checklists, and investigation notes tied to specific experiments run against the Airport mezzanine DXF.
- [`research/`](research/) — the broader representation thesis, research framing, and implementation sketches around the take-home.
- [`process/`](process/) — task-progress and context: the session journal, layer etymology, layer-normalization analysis, REPL approach plan, and the file map.

None of the files here are imported by the pipeline. They are the record of how we got to the current `augrade/` package and why the tradeoffs look the way they do.
